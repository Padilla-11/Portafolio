Simulador Epidemiológico SIR

Modelado matemático de enfermedades infecciosas

Simulador Epidemiológico SIR

Simulador interactivo de propagación de enfermedades infecciosas utilizando el modelo matemático SIR (Susceptibles-Infectados-Recuperados) con ecuaciones diferenciales y métodos numéricos de Runge-Kutta y Euler.

HTML5 CSS3 JavaScript Chart.js

Características principales

  • Implementación de ecuaciones diferenciales del modelo SIR
  • Métodos numéricos: Runge-Kutta 4 y Euler
  • Simulación de enfermedades: COVID-19, Dengue, Fiebre Amarilla
  • Cálculo del número reproductivo básico (R₀)
  • Validación con datos reales del Departamento del Cesar
  • Análisis de sensibilidad de parámetros
  • Diagramas de fase y gráficos temporales interactivos
  • Exportación de datos en CSV y gráficos en PNG

Tecnologías utilizadas

  • JavaScript vanilla para cálculos numéricos
  • Chart.js para visualización de datos
  • HTML5 y CSS3 para interfaz responsive
  • Algoritmos de optimización para ajuste de parámetros
  • Procesamiento de archivos CSV
  • Proyecto académico - Universidad Popular del Cesar
